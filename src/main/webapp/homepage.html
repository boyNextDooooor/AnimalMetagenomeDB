The development of high-throughput sequencing stimulates a series of microbiome technologies, such as amplicon sequencing, whole genome sequencing, metagenome, metatranscriptome, which have rapidly promoted the development of the microbiome field. In recent years, the number of metagenomic data in public repositories is increasing exponentially. Here, we introduce the AnimalMetagenome DB, which contains all public animal microbiome data, divided into domestic animal and wild animal. AnimalMetagenome DB integrates microbiome sequencing data with host information, making it easier for researchers to find animal metagenomic data of interest. Currently, AnimalMetagenome DB version 1.0 includes microbiome data of pig, bovine, horse, sheep and about wild animals. These microbiome data cover metagenomes metadata, countries, projects. Researchers can browse and search for animal metagenomics data of interest by selecting animal species, sample site, study type, sequencing platform, etc.
